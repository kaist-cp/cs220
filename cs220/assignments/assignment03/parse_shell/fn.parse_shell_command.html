Parse the string as a shell command.

Usually, a shell command is whitespace-separated array of strings.

cat file  -->  ["cat", "file"]

But sometimes, you may want to include whitespaces in each argument. In that case, you can use quotes.

ls 'VirtualBox VMs'  -->  ["ls", 'VirtualBox VMs']
ls VirtualBox' 'VMs  -->  ["ls", 'VirtualBox VMs']

For simplicity, you may assume that the string only contains alphanumeric characters, spaces (“ “), and single quotes (”’“).

See test_shell for more examples.
